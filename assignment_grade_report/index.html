Canvas admin utility

Assignment Grade Report

Install one bookmark, choose enrollment terms and courses in Canvas, and download assignment-level grades as a CSV.

Quick installation

Drag it once. Use it whenever you need it.

The bookmarklet loads the current report script from GitHub Pages, so future script updates are available without reinstalling the bookmark.

Drag this button to your bookmarks bar Canvas Grade Report

Do not click the button on this page. Drag it onto the bookmarks bar, then click it while you are signed in to Canvas.

Don’t see the bookmarks bar?

Windows or ChromeOS: Ctrl + Shift + B

macOS: ⌘ + Shift + B

You can use the same bookmark in Chrome, Edge, Firefox, or Safari.

Manual installation

Create a new bookmark named Canvas Grade Report, then paste the copied bookmarklet into the bookmark’s URL or Location field.

Report workflow

How to use it

  1. Open Canvas Sign in and open any normal Canvas page, such as the Dashboard.
  2. Launch the report Click Canvas Grade Report in your bookmarks bar and wait for the terms to load.
  3. Choose the scope Both named subaccounts are selected by default; keep either one or both, enter at least three course-code characters, and select the enrollment terms.
  4. Review matched courses Click Find Courses, then keep only the course sections you want included.
  5. Generate the CSV Click Generate Report. Keep the Canvas tab open until the completion message appears.
  6. Open the download The CSV downloads automatically and contains one row per student assignment record.

Before you begin

Good to know

Canvas permissions apply

You can only report grades and courses your Canvas account is authorized to view.

Works on your current Canvas site

The same bookmark can run in production, beta, or test Canvas environments. Account names are looked up when the tool opens.

Large reports take time

Canvas paginates grade data. The progress panel shows active courses, requests, rows, and elapsed time.

Read-only report

The tool reads Canvas data and creates a local CSV. It does not change grades or course settings.

GitHub Pages hosts the JavaScript file. Canvas report data stays in your browser and the CSV it creates; the tool does not upload grade data to GitHub.

Troubleshooting

If something doesn’t work

Nothing happens when I click the bookmark

Confirm that the current tab is a Canvas page and that the bookmark’s URL begins with javascript:. Institution-managed browser settings may disable bookmarklets. If necessary, open the source link below, copy the script, and run it from the browser developer console while Canvas is open.

No terms or courses are returned

Check your Canvas permissions, use at least three course-code characters, and verify that the selected subaccounts and enrollment terms are correct. The course search checks the chosen account scopes and their descendant subaccounts.

The CSV does not download

Keep the report window open until it says Report complete and allow downloads from your Canvas domain. If a run is canceled after collecting rows, use Download Partial Results in the report window.
